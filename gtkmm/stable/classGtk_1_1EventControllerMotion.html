Detailed Description

Event controller for motion events.

Gtk::EventControllerMotion is an event controller meant for situations where you need to track the position of the pointer.

Since gtkmm 3.94:

Glib::PropertyProxy_ReadOnly< bool > Gtk::EventControllerMotion::property_contains_pointer ( ) const

Whether the pointer is in the controllers widget or a descendant.

See also Gtk::EventControllerMotion::property_is_pointer().

When handling crossing events, this property is updated before Gtk::EventControllerMotion::signal_enter() but after Gtk::EventControllerMotion::signal_leave() is emitted.

Default value: false
