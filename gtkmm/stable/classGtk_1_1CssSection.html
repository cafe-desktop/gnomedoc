Detailed Description

Defines a part of a CSS document.

Because sections are nested into one another, you can use get_parent() to get the containing region.

A CssSection is input data to a Gtk::CssProvider::signal_parsing_error() handler.

Glib::RefPtr<CssSection> Gtk::CssSection::get_parent ( )

Gets the parent section for the given section.

The parent section is the section that contains this section. A special case are sections of type Gtk::CssSection::Type::DOCUMENT. Their parent will either be nullptr if they are the original CSS document that was loaded by Gtk::CssProvider::load_from_file() or a section of type Gtk::CssSection::Type::IMPORT if it was loaded with an import rule from a different file.

Returns
The parent section or nullptr if none.
