Description

GspellEntryBuffer extends the GtkEntryBuffer class with spell checking support.

Functions

gspell_entry_buffer_set_spell_checker ()

void
gspell_entry_buffer_set_spell_checker (GspellEntryBuffer *gspell_buffer,
                                       GspellChecker *spell_checker);

Sets a GspellChecker to a GspellEntryBuffer. The gspell_buffer will own a reference to spell_checker , so you can release your reference to spell_checker if you no longer need it.

Parameters

gspell_buffer

a GspellEntryBuffer.

  

spell_checker

a GspellChecker, or NULL to unset the spell checker.
