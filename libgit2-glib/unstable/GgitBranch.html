ggit_branch_delete ()

void
ggit_branch_delete (GgitBranch *branch,
                    GError **error);

Deletes an existing branch reference.

If the branch is successfully deleted, this object is not useful anymore and if should be freed with g_object_unref().

a GgitBranch.
