Glade User Interface Designer Reference Manual

for gladeui glade 3.36.0 . This document attempts to describe how to use the Glade UI Designer core library to integrate Glade into your application and how to integrate your custom GTK+ derived widgets into the Glade UI Designer. The latest version of this documentation can be found on-line at https://developer.gnome.org/gladeui/stable .
