GTK+ 3 Reference Manual

This document is for the GTK+ 3 library, version 3.24.29 . The latest versions can be found online at https://developer.gnome.org/gtk3/. If you are looking for the older GTK+ 2 series of libraries, see https://developer.gnome.org/gtk2/.
